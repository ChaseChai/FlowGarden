# 「镜园」生活规划秘书 APP 方案 v4
## —— 你的数字人秘书，住在一座会呼吸的花园里

> **赛道定位**：10 蓝盒子（Hack the Rest）× 02 清闲智能（Desktop Daemon）
> **产品形态**：生活规划秘书 APP（移动端 + 桌面常驻）
> **感知硬件**：语音戒指 / 手环（睡眠 + 专注 + 活动检测）
> **核心概念**：一个以数字人为中心的生活规划秘书——它读懂你的睡眠、专注与状态，替你规划每一天；它住在艺术花园里，花园和它，都是你自身的映射。

## 三、数字人：秘书 + 映射的双重角色

### 3.1 作为生活秘书（功能人格）

| 时刻 | 秘书行为 | 示例台词 |
|------|----------|----------|
| **早晨** | 晨间简报 | "早安。昨晚深睡只有 1 小时，今天给你排的 3 个任务里，我把最难的挪到了下午你状态最好的时段。上午先处理轻松的邮件吧。" |
| **上午** | 专注陪伴 | "接下来 45 分钟是专注时间，我陪你。我在看这本书。"（数字人在花园里做自己的事） |
| **午后** | 状态提醒 | "你已经坐了 2 小时了，手环显示你有点紧绷。要不要起来走走？我去给花园浇水等你。" |
| **傍晚** | 晚间复盘 | "今天完成了 4 件事，分心了 3 次——不过你都诚实标记了，很棒。明天的日程要现在看看吗？" |
| **睡前** | 入睡仪式 | "明天 8 点有个会，我们 11 点睡好吗？现在跟我做三轮深呼吸……" |

### 3.2 作为自身映射（情感人格）

| 你的状态 | 数字人状态 | 花园氛围 |
|----------|------------|----------|
| 睡眠充足 + 专注高效 | 神采奕奕，动作轻快 | 阳光明媚，花开 |
| 睡眠不足 | 打哈欠，揉眼睛，动作迟缓 | 晨雾未散 |
| 长时间工作 | 疲惫，趴在桌上 | 黄昏色调 |
| 连续分心焦虑 | 安静地坐在角落 | 阴天微雨 |
| 完成目标 | 开心转圈，撒花 | 彩虹/花瓣雨 |
| 长期自律 | 解锁新衣服/新动作 | 花园新区域 |

**关键设计原则**：数字人不是打卡机器，是"另一个你"——它不会说"你又失败了"，而是"我们今天都有点累，早点休息吧"。

## 四、花园：艺术氛围层（轻量方案）

### 4.1 设计原则：好看 > 技术

- **不做**精细 L-system 算法生成
- **要做**高质量美术资源 + 状态驱动的场景切换
- 类似《旅行青蛙》《Tsuki 月兔冒险》的治愈系场景逻辑

### 4.2 多画风皮肤（可切换/可解锁）

| 画风 | 风格参考 | 氛围 |
|------|----------|------|
| 水彩花园 | 吉卜力/《Gris》 | 柔和治愈 |
| 像素庭院 | 《星露谷物语》 | 复古游戏感 |
| Low Poly 山谷 | 《纪念碑谷》 | 极简几何 |
| 国风庭院 | 水墨 + 留白 | 东方禅意（契合"中国文化出海"方向） |
| 赛博温室 | 霓虹 + 全息植物 | 未来感 |

### 4.3 状态 → 场景映射（轻量实现）

```javascript
// 花园 = 预渲染场景层 + 天气粒子层 + 时间光照层
const GardenScene = {
```

## 五、戒指/手环检测（轻量集成）

### 5.1 三模式检测

| 模式 | 检测内容 | 用途 |
|------|----------|------|
| **睡眠模式**（夜间自动） | IMU 体动 → actigraphy 分期（深/浅/REM/醒） | 晨间简报、精力预测 |
| **专注模式**（工作时段） | IMU 静止+微颤 → 专注/分心 | 专注陪伴、番茄统计 |
| **日常模式**（全天） | 活动量、久坐检测 | 活动提醒、状态推断 |

### 5.2 交互（保持简单）

| 输入 | 作用 |
|------|------|
| 双击 | 开始/结束专注 |
| 拍击 | 标记分心 / 快速记录"我现在状态不好" |
| 手势 | （可选）翻页看简报 / 召唤数字人 |
| 语音 | 对秘书说话："帮我排一下明天" / "今天不想工作" |

### 5.3 硬件解耦设计

## 六、APP 功能地图

```
镜园 APP
├── 首页：数字人 + 花园（主界面，一眼看到"它和今天的你"）
├── 今日：晨间简报 → 日程时间线 → 专注时段 → 晚间复盘
├── 规划：日程管理 / 习惯目标 / 智能排程（LLM 建议）
├── 数据：睡眠报告 / 专注趋势 / 状态-效率关联
├── 花园：画风皮肤 / 解锁装饰 / 好友探访（可选社交）
└── 设置：设备连接（戒指/手环）/ 通知 / 隐私
```

### 核心闭环

```
睡眠检测 → 晨间简报（状态+建议日程）
    → 日间执行（专注+提醒+秘书陪伴）
    → 晚间复盘（总结+明日规划）
    → 入睡仪式（呼吸引导）
    → 睡眠检测（验证改善）
    ↺ 循环：每周生成生活洞察周报
```
