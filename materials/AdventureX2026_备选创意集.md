# AdventureX 2026 备选创意集（推翻重来版）

> 筛选标准：**视觉冲击力第一**，戒指交互自然，48h-1 周可实现，契合赛道 10（休息/睡眠）/ 02（Desktop Daemon）/ 03（Vibe）
> 全部推翻「花园 + 数字人秘书」框架，六个全新方向。

---

## Idea 1：「心流墨境」—— 你的每一天，是一幅山水画

**一句话**：网页是一幅实时渲染的国风水墨长卷——你的专注是沉稳的笔触，分心是飞溅的墨点，睡眠是远山，一天结束生成一幅独一无二的"心境山水"，可分享可收藏。

| 维度 | 设计 |
|------|------|
| **视觉核心** | WebGL 水墨渲染：墨色晕染、留白、飞白笔触。画面随时间从右往左"长"出一幅长卷 |
| **状态映射** | 深度专注 = 中锋重墨（山）；浅专注 = 淡墨（雾）；分心 = 墨点飞溅；睡眠好 = 次日画卷右下角多一轮日出 |
| **戒指交互** | 手势 rotate = 换墨色（焦/浓/重/淡/清）；拍击 = 钤印（盖一颗朱砂印章，标记重要时刻）；语音 = 题字（"今日诸事顺遂"→ AI 书法落款） |
| **Demo 画面** | 评委盯着空白宣纸，选手专注 1 分钟，远山缓缓浮现；选手故意抖手，墨点炸开；最后 AI 题字落款，**一幅完整的画当场诞生** |
| **赛道** | 10 + 03 Vibe + 米哈游/文化出海（国风是天然差异化） |
| **实现难度** | 低-中：WebGL shader 水墨有开源参考（如 shadertoy 大量墨染效果），核心是"状态→笔触参数"的映射 |
| **杀手锏** | **每日一幅画 = 天然社交货币**，用户会主动发朋友圈/小红书 |

## Idea 2：「情绪流体」—— 搅动你内心的那团火

**一句话**：屏幕中央是一团绚烂的 WebGL 流体（星云/极光质感），它是你内心状态的实时具象——戒指手势可以直接"搅动"它，睡眠和专注数据决定它的底色与粘度。

| 维度 | 设计 |
|------|------|
| **视觉核心** | WebGL Fluid Simulation（有成熟开源库）：彩色流体 + 光晕 + 粒子，视觉冲击最强的一档 |
| **状态映射** | 心流 = 金色稳定涡旋；焦虑 = 暗红湍流；疲惫 = 深蓝缓慢流动；睡眠好 = 次日流体里多一层星光 |
| **戒指交互** | 挥手 = 搅动流体（IMU 速度→流体速度）；画圈 = 制造涡旋；拍击 = 流体炸开又聚合（释放压力）；语音 = "我想看海"→ 切换蓝色系 |
| **Demo 画面** | 选手对着屏幕挥手，**流体跟着手部轨迹翻卷炸开，全场"哇"出声**；然后展示"焦虑的一天"vs"心流的一天"两段流体录像对比 |
| **赛道** | 03 Vibe + 10 + 05 智能少年（自我表达） |
| **实现难度** | 低：fluid simulation 库开箱即用，工作量全在"状态→流体参数"调优 |
| **杀手锏** | **交互快感最直接**——戒指第一次让你"摸到"自己的情绪 |

## Idea 3：「睡眠地下城」—— 每晚睡着后，冒险才开始

**一句话**：每晚你的睡眠是一场 Roguelike 地下城冒险——深睡是向下探索的钥匙，夜醒是遭遇怪物，翻身是陷阱；早晨数字人讲述"你昨晚的冒险战报"，睡眠结构变成游戏地图。

| 维度 | 设计 |
|------|------|
| **视觉核心** | 像素风/Low Poly 地下城俯视图：火把光晕、迷雾驱散、角色行走动画。早晨的"冒险回放"是核心视觉 |
| **状态映射** | 入睡 = 进入地下城；深睡 = 下到更深一层（B1→B5）；REM = 开宝箱（梦境）；夜醒 = 遇怪掉血退回上层；整晚安睡 = 见到最终 Boss"晨光巨龙" |
| **戒指交互** | 夜间：体动驱动冒险（actigraphy）；早晨：拍击 = 开宝箱看战利品；手势 = 选择今晚的"冒险天赋"（"今晚我想深睡"+22:30 提醒入睡） |
| **Demo 画面** | 直接播放"昨晚冒险回放"：小人举着火把下楼梯，凌晨 3 点遇怪退回（对应真实夜醒数据），早晨战报"深睡 58 分钟 = 到达 B3 层，击败 2 只怪物"。**评委看完就懂什么叫睡眠结构** |
| **赛道** | 10（正中靶心）+ 米哈游游戏化 + 08 B站（游戏区天然内容） |
| **实现难度** | 中：睡眠分期算法 + 游戏地图生成规则，美术可用现成像素素材 |
| **杀手锏** | **把最枯燥的睡眠数据变成最想看的"昨晚发生了什么"**——用户第二天睁眼第一件事就是看战报 |

## Idea 4：「桌面生态缸」—— 你屏幕角落养着一缸活物

**一句话**：你的电脑桌面角落有一个永远运行的生态缸/水族箱——鱼群是你的任务，水质是你的身体状态，睡眠好水就清澈，久坐水就浑浊；戒指是你与这缸小世界互动的手。

| 维度 | 设计 |
|------|------|
| **视觉核心** | 精致 2D 水族箱：光影水波、摇曳水草、不同鱼种（每条鱼=一个任务/习惯），丁达尔光效，治愈系天花板 |
| **状态映射** | 睡眠好 = 水质清澈阳光透入；久坐 = 水渐浑浊；完成任务 = 鱼长大/繁殖；连续自律 = 新鱼种/珊瑚入住；熬夜 = 缸壁长藻 |
| **戒指交互** | 手势 = 喂食/捞垃圾/敲缸壁（鱼群受惊散开）；拍击 = 快速查看每条鱼对应的任务；语音 = "放一条新鱼"（新建任务） |
| **Demo 画面** | 分屏对比：左边"健康作息一周的缸"（清澈、鱼群丰茂），右边"连续熬夜的缸"（浑浊、鱼无精打采）。**无需一句话，评委就懂了这个产品的全部逻辑** |
| **赛道** | 02 Desktop Daemon（正中靶心：桌面常驻精灵）+ 10 + 07 小红书（晒缸） |
| **实现难度** | 低-中：2D 骨骼动画鱼群 + 水波 shader，素材生态成熟 |
| **杀手锏** | **Desktop Daemon 的完美形态**——它不占你注意力，但你总想瞄一眼"我的缸今天怎么样" |

## Idea 5：「时间织锦」—— 把你的日子织成一幅挂毯

**一句话**：网页里有一台古老的织机，你的每一天是一根线——深睡是靛蓝，心流是金色，分心是灰白，休息是月白；一个月织完，你的生活是一幅可以挂满墙的织锦。

| 维度 | 设计 |
|------|------|
| **视觉核心** | 织机动画 + 织物纹理实时生长：梭子来回、经线绷紧、花纹浮现。拉远看是挂毯，拉近看每一行是一天 |
| **状态映射** | 24 小时 = 一行织物；每个小时的纬线颜色 = 该时段状态（深睡/浅睡/专注/分心/活动/离线）；特殊日子织入金线图案（连续 7 天好眠 = 一朵纹样） |
| **戒指交互** | 手势 rotate = 推动梭子（手动"织入"今天的数据）；拍击 = 在某行打结（标记纪念日）；语音 = "上个月怎么样"→ 织锦自动滚动画外音解读 |
| **Demo 画面** | 镜头先怼在织机上，梭子随选手演示来回穿梭织出两行；然后**镜头猛拉远，一面墙挂满 30 天的织锦，金蓝灰的纹理清晰可见哪周自律哪周混乱**——数据可视化的终极浪漫 |
| **赛道** | 10 + 03 + 08 B站（"我的一年织成一幅毯子"必火选题） |
| **实现难度** | 低：本质是热力图的艺术化封装，织机感动画 + Canvas 纹理 |
| **杀手锏** | **长期价值最强**：用户用一年，就有一幅真实的、属于自己生活的艺术品（甚至可以实体打印） |

## Idea 6：「呼吸房间」—— 整个空间随你一起呼吸

**一句话**：网页是一间 3D 房间，它随你的呼吸节律起伏——墙壁吸气时微微膨胀，呼气时缓缓回落；你的状态改变房间的陈设与光线，做呼吸练习时，整个房间与你同频共振。

| 维度 | 设计 |
|------|------|
| **视觉核心** | 3D 空间沉浸感：房间整体"呼吸"（缩放+光照脉动），家具、烛火、窗外景色随状态变化；呼吸引导时整个空间成为你的节拍器 |
| **状态映射** | 焦虑 = 房间逼仄昏暗杂物堆积；平静 = 宽敞明亮烛火温暖；深度心流 = 房间化作星空下的小屋；睡眠中 = 房间也"睡着"（缓慢深长的呼吸起伏） |
| **戒指交互** | IMU 检测呼吸节律 → 房间同步起伏；手势 = 推开窗（深呼吸模式）/ 点亮蜡烛（冥想模式）；拍击 = 切换窗外景色（海/山/雨/雪） |
| **Demo 画面** | 选手平躺做 4-7-8 呼吸，**整个房间跟着选手的呼吸膨胀、回落、膨胀、回落，烛火随呼气摇曳**——评委下意识跟着调整了自己的呼吸 |
| **赛道** | 10（呼吸/冥想/入睡仪式）+ 02 + 09 Dimensional（空间感） |
| **实现难度** | 中：Three.js 房间 + 呼吸节律检测（IMU 微动提取），场景不需复杂建模 |
| **杀手锏** | **第一个"有体感"的网页**——呼吸是人类唯一能自主控制的自主神经功能，房间与它同步就是最直接的人机共频 |
